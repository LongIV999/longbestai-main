# Content Style Guide

## Copy Rules

### FB/Zalo Captions
- **Plain text only** - NO markdown headings (`# `)
- Visual-first approach: 80% visual, 20% text
- Must include CTA with all 3 elements (see CTA Rules below)

### Blog Posts
- Plain text body (no markdown yet)
- Vietnamese language preferred
- Clear, conversational tone
- Focus on practical value

### Deals
- Clear pricing with cost/retail transparency
- Include duration, warranty, inventory when applicable
- Highlight special conditions or limitations

### Courses
- Vietnamese language
- Clear learning outcomes (minimum 1)
- Structured lessons (minimum 4)
- Beginner-friendly explanations

## CTA Rules (Mandatory Format)

Every piece of content **MUST** include ALL 3 elements:

1. **Relevant keyword** (tên sản phẩm/dịch vụ)
2. **Zalo**: 0903 469 888
3. **Group link**: https://zalo.me/g/uwawns336

### CTA Examples

✅ **Good**:
```
Đặt hàng OpenClaw Setup ngay! Zalo 0903 469 888 hoặc tham gia nhóm https://zalo.me/g/uwawns336
```

✅ **Good**:
```
Liên hệ tư vấn AI automation qua Zalo 0903 469 888 hoặc tham gia nhóm https://zalo.me/g/uwawns336
```

❌ **Bad** (missing group link):
```
Liên hệ Zalo 0903 469 888
```

❌ **Bad** (missing Zalo):
```
Tham gia nhóm: https://zalo.me/g/uwawns336
```

## Visual-First Rules (Jobs)

### Layout Principles
- **Minimal text on images**: 80% visual, 20% text
- **Clear focal point**: Guide eye with overlay elements
- **Brand consistency**: Use brand colors and themes

### Overlay Elements
- **Maximum 3-5 items** to avoid clutter
- Types: stickerRect, arrow, highlightScribble, icon
- Strategic placement to highlight key info

### Example Overlay Strategy
```json
[
  { "type": "stickerRect", "text": "TUYỂN DỤNG" },  // Main headline
  { "type": "highlightScribble" },                    // Emphasize salary/benefit
  { "type": "arrow" },                                 // Direct attention
  { "type": "icon", "iconName": "star" }              // Visual accent
]
```

## Brand Colors

### Primary Accent
- **#00F0F0** (cyan)
- Use for: primary CTAs, headings, highlights
- Dominance: 40-60% of color usage

### Lime Highlight
- **#B6FF00** (bright lime)
- Use for: accents, highlights, emphasis
- **IMPORTANT**: Limit to 5-15% usage only (sparingly!)
- Never use as background color

### Usage Guidelines

✅ **Good**:
- Background: #00F0F0
- Small highlight: #B6FF00 (10% of design)

❌ **Bad**:
- Background: #B6FF00 (too aggressive)
- Overuse: 50% lime, 50% cyan (unbalanced)

## Theme Recommendations

### bridge-brutal (Recommended for Jobs)
- High contrast, bold typography
- Perfect for attention-grabbing job cards
- Pairs well with #00F0F0 and #B6FF00

### official-light
- Clean, professional
- Good for blog posts and deals

### dark-aurora
- Modern, tech-focused
- Good for product showcases

## Tone & Voice

### Brand Personality
- **Professional** yet approachable
- **Knowledgeable** without being condescending
- **Action-oriented** - focus on outcomes

### Writing Style
- Short sentences, clear paragraphs
- Active voice preferred
- Use Vietnamese idioms naturally
- Avoid jargon unless explaining technical concepts
- Include examples when possible

## Content Checklist

Before publishing any content:

- [ ] Contains all 3 CTA elements (keyword + Zalo + group link)
- [ ] Uses brand colors appropriately (#00F0F0 primary, #B6FF00 ≤15%)
- [ ] Job captions are plain text (no `# ` headings)
- [ ] CapCut deals include livestream warning
- [ ] Visual-first jobs have 3-5 overlay items max
- [ ] Tone is professional yet approachable
- [ ] All prices/terms are accurate and clear
